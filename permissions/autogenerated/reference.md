## Default Permission

Default permissions for the Android Battery Optimization plugin. Allows all battery optimization commands.

#### This default permission set includes the following:

- `allow-check-battery-optimization-status`
- `allow-request-battery-optimization-exemption`
- `allow-open-battery-settings`
